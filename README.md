Steps of Python Working
Write the Python Program
Write your code in a file with the .py extension.
Example: hello.py
Python Interpreter Reads the Code
The Python interpreter reads your program line by line.
Converts to Bytecode
Python converts the source code into bytecode (an intermediate form).
Python Virtual Machine (PVM) Executes It
The Python Virtual Machine (PVM) executes the bytecode.
Output is Displayed
The result is shown on the screen.
